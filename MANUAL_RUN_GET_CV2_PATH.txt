# 手动运行get_cv2_path.py脚本指南

由于PowerShell中路径包含特殊字符导致自动运行失败，请按照以下步骤手动运行脚本：

## 步骤1: 打开命令提示符
- 按下Win + R键打开运行窗口
- 输入"cmd"并按Enter键

## 步骤2: 切换到项目目录
- 复制以下命令并粘贴到命令提示符中，然后按Enter键：
  `cd /d "d:\新建文件夹 (2)\cv2"`

## 步骤3: 运行批处理文件
- 输入以下命令并按Enter键：
  `run_get_cv2_path.bat`

## 步骤4: 记录输出信息
- 脚本会显示cv2模块的安装路径和相关dll文件信息
- 请记录这些信息，以便在main.spec文件中正确配置cv2模块

## 注意事项
- 请确保路径中的引号是英文双引号
- 如果您遇到任何问题，请检查路径是否正确

祝您成功！
